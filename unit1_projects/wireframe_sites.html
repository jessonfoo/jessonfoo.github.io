Identifying visual challenges, and problems

Uncrate

The goal of this website is to allow users to explore new and cool products. It is geared towards a generally young audience. The design is meant to be simple and draws attention to colorful visuals of an item being described.

Reddit

Reddit solves the problem of searching for interesting content. The design allows for users to see other user submissions that are trending.logging in allows for the user to rate submissions.

Impossible Cool Tumblr

Impossible cool uses tumblr's general layout. The header is composed of the title of the main page with link, the content is mostly images and a footer at the bottom. Tumblr's layout is visually engaging with large images, small text and simple links. It's primary purpose is to entertain users with engaging images often with large stylized or embedded text.
